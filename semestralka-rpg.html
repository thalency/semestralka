Úvod

Semestrální práce z předmětu Značkovací jazyky se zabývá tvorbou XML dokumentu popisujícího soubor do kterého aplikace ukládá svá nastavení. Součástí je také tvorba XSD schéma, XSL transformace do JSON a tato dokumentace. Dokument byl vytvořeno jako konfigurace postavy v RPG mobilní hře.

Metodika

Všechny dokumenty byly vytvořeny v programu NetBeans IDE 8.2 a SublimeText pro MacOS.

Nejprve byl vytvořen hrubý XML dokument se jmény elementů a atributů. Ten byl následně konverttován do XSD pomocí webové aplikace Freeformater v designu Russian Doll.

Finální XML bylo vytvořeno ručně.

XSL transformace do JSON byla psána ručně.

Celý projekt byl nahrán na GitHubu.

Popis transformace

JSON

Data jsou transformována do JSON dokumentu. Jsou strukturována stejně jako výchozí XML soubor.

Závěr

Jazyk XML je používán pro strukturované ukládání dat nebo při komunikaci mezi programy. Jeho výhodou je dobrá čitelnost a šikoké možnosti filtrování a omezování dat a datovýh typů. Práce s ním je také snadná, protože k jeho editaci stačí pouze textový editor.
